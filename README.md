# echo.ml

Echo server written in OCaml with LWT.

## Build

```
jbuilder build
```

## Run

### 1. Launch Server

```
./_build/default/echo.exe
```

### 2. Connect to server using telnet

```
telnet localhost 8080
ping
```

### 3. Server returns response

```
pong
```
